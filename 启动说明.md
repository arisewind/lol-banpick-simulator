# 🎮 LoL Ban/Pick Simulator - 快速启动

## 双击运行以下脚本即可启动应用：

### 📦 启动脚本说明

| 脚本文件 | 用途 | 说明 |
|---------|------|------|
| **启动开发环境.bat** | 🔥 **推荐日常使用** | 启动完整的 Electron 应用（Vite + Electron 窗口） |
| **快速启动（仅前端）.bat** | 🌐 仅前端开发 | 只启动 Vite 开发服务器，浏览器访问 `http://localhost:5173` |
| **构建并运行.bat** | 📦 构建生产版本 | 构建 Windows 安装包到 `release\` 目录 |

---

### 🚀 首次运行

1. **双击** `启动开发环境.bat`
2. 首次运行会自动安装依赖（需要等待几分钟）
3. 安装完成后自动启动应用

---

### 💡 开发调试

- **修改代码后自动刷新**：使用 `启动开发环境.bat`
- **只测试 UI 样式**：使用 `快速启动（仅前端）.bat`，在浏览器中更快预览
- **构建可分发的 .exe**：使用 `构建并运行.bat`

---

### 🛠️ 故障排除

#### 问题：`pnpm` 命令不存在
**解决**：安装 pnpm 包管理器
```bash
npm install -g pnpm
```

#### 问题：Electron 窗口无法启动
**解决**：检查 `node_modules/electron/dist/electron.exe` 是否存在，如果缺失：
```bash
pnpm install --force
```

#### 问题：端口 5173 被占用
**解决**：关闭其他使用该端口的程序，或修改 `vite.config.ts` 中的端口配置

```
lol banpick simulator/
├── 启动开发环境.bat        ← 双击这个！

```

---

### ⌨️ 命令行方式

如果你想使用命令行：

```bash
# 开发模式（Electron 窗口）
pnpm electron:dev

# 仅前端开发服务器
pnpm dev

# 构建安装包
pnpm electron:build
```
